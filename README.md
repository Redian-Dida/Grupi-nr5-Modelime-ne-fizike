# Projekti: Modelime në Fizikë - Detyra 1

### Anëtarët e Grupit:
1.Redian Dida
2.Klevio Shtembari
3.Eglant Lacka

### Qëllimi i projektit
Ky projekt demonstron përdorimin e Python për llogaritje shkencore, duke përdorur libraritë NumPy për analizë numerike dhe Matplotlib për vizualizimin e të dhënave.

### Përshkrimi i figurës
Figura paraqet një funksion determinist që kombinon një valë sinusoidale me një shuarje eksponenciale dhe një zhurmë periodike, duke simuluar një sinjal fizik real.
