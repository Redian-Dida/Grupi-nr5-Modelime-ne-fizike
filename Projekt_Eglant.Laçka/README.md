# Eksperimenti 3

Ky projekt përmban një skript Python për integrimin numerik të dy oscilatorëve të coupled dhe vizualizimin e energjisë në funksion të kohës.

## Strukturë

- `src/eksperimenti_3.py` - skripta kryesore Python
- `requirements.txt` - varësitë Python të nevojshme
- `figures/` - dosja ku ruhet figura e krijuar automatikisht

## Si të ekzekutoni

1. Instaloni varësitë:
   ```bash
   python -m pip install -r requirements.txt
   ```
2. Ekzekutoni skriptën:
   ```bash
   python src/eksperimenti_3.py
   ```
3. Figura rezultuese ruhet në `figures/eksperimenti_3.png`.
